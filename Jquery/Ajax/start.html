동기- 반드시 순서대로 실행. 앞의 것이 끝나야만 뒤에 것을 실행 비동기- 동시에도 여러가지 명령을 수행 가능. 순서도 상관없음. [javascript 분파] -Node.js : 백엔드에서도 쓰이는 언어 -react : 프론트엔드를위한언어 -Vue.js : 프론트엔드를위한언어 fetch : javascript에서 제공(IE11에서 실행안되는 치명적 단점 있음) Axios : node.js 에서 제공 Ajax : Jquery에서 제공 //이런 비동기 방식을 연동하는 이유:백엔드에서 데이터를 받기 위한 방법 *비동기방식 쓰는 이유 XML, JSON(JavaScript Object Notation), 텍스트, HTML 등의 정보를 기다림없이 교환하기 위해. API(Application Programming Interface 애플리케이션 프로그래밍 인터페이스, 응용 프로그램 프로그래밍 인터페이스) 응용 프로그램에서 사용할 수 있도록, 운영 체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스를 뜻한다. 서버에 누군가 만들어놓은 데이터를 손쉽게 받을수 있는 무언가.
