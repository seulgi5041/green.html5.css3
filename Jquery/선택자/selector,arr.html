탐색 선택자

인덱스사용: 위치 탐색 선택자

firs/last

even(짝수)/odd(홀수) : index를 기준 0도 짝수로 인식한다. (짝수 인덱스, 홀수 인덱스)

eq(내가 지정한 배열!) / It(내가 지정한 배열보다 작은) / gt(내가 지정한 배열보다 큰)

fist-of-type / last-of-type (각각의 무리 중, 그룹 중)

slice

only-chile : 그룹들 중 자식이 하나인 tag만 선택한다. 자식이 여러개인 그룹은 선택하지 않음.

nth-child / nth-last-of-type (단순 숫자로)






배열관련메서드

each() / $.each() : for문과 비슷하다. 태그들을 각각 뽑는 기능

$.map()/$.grep() : each와 쓰는 법 동일하나, return 값을 가짐.

$.map() : 내가원하는 데이터를 가공하여 데이터자체를 리턴

$.grep() : 내가 원하는 데이터를 가공하여 true/false 값으로 리턴

$.inArray : 배열 데이터 중 지정한 데이터의 인덱스 값을 반환 / $.isArry: 배열객체일때 true / 배열객체 아니면 false(*Json배열은 객체가 아님) / $.merge : 두 배열 객체를 하나의 객체로 묶음






속성 탐색 선택자

$("요소선택[속성]") - [속성] 포함한 요소를 선택

$("#wrap a[targer]") id가 wrap요소인 a태그에서 taget속성만 선택


$("요소선택[속성^=값]") - [속성값]으로 시작하는 요소 선택

$("#wrap a[href^-https]") id가 wrap요소인 a태그에서 https로 시작하는 요소 선택


$("요소선택[속성$=값]") - [속성값]으로 끝나는 요소 선택

$("#wrap a[href$=net]") id가 wrap요소인 a태그에서 net로 끝나는 요소만 선택


$("요소선택[속성*=값]") - [속성값]이 포함된 요소를 선택

$("#wrap a[href*=google]") id가 wrap요소인 a태그에서 goolgle가 포함된 속성만 선택


$(":type 속성값") - 속성값과 일치하는 요소만 선택

$("#member_f :password") id가 member_f 요소에서 속성값이 password인 것만 선택
